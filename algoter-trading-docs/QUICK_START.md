## ðŸ“– Your First Strategy

### Step 1: Connect Your Wallet

1. Click "Get Started" on the home page
2. Connect your Phantom wallet
3. You'll be redirected to the Studio

### Step 2: Build a Strategy

1. **Open Studio**: Navigate to `/studio`
2. **Add Blocks**: Drag blocks from the sidebar to the canvas
   - **Indicators**: Moving Average, RSI, MACD
   - **Conditions**: Price Above, Crossover
   - **Actions**: Buy, Sell, Set Stop Loss
   - **Risk Management**: Position Size

3. **Configure Blocks**: Click on a block to edit its settings

4. **View Code**: Switch to the "Code" tab to see generated code

### Step 3: Backtest Your Strategy

1. Click "Run Backtest" in the Studio
2. Configure backtest parameters:
   - Start Date: 1/1/2025
   - End Date: Today
   - Initial Capital: $10,000
3. View results:
   - Performance metrics
   - Trade history
   - Equity curve

### Step 4: Monitor Markets

1. Go to **Trading Dashboard** (`/trading`)
2. View real-time market data
3. Get AI strategy advice
4. Monitor Twitter accounts
