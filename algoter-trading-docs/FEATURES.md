## üé® Strategy Builder

### Drag & Drop Interface
- Visual block-based strategy creation
- No coding required
- Real-time code generation

### Block Types

#### Indicators
- **Moving Average (SMA/EMA)**: Trend following indicator
- **RSI**: Relative Strength Index for overbought/oversold conditions
- **MACD**: Moving Average Convergence Divergence
- **Bollinger Bands**: Volatility indicator

#### Conditions
- **Price Above/Below**: Compare price to a value
- **Crossover**: Detect when indicators cross
- **Percent Change**: Monitor price movements

#### Actions
- **Buy**: Execute buy orders
- **Sell**: Execute sell orders
- **Set Stop Loss**: Risk management
- **Set Take Profit**: Lock in profits

#### Risk Management
- **Position Size**: Control trade size
- **Max Drawdown**: Limit losses
- **Portfolio Allocation**: Diversify positions

### Code Generation
- Automatic code generation from blocks
- Structured code with `initialize()`, `onBar()`, `onExit()`
- Copy to clipboard functionality
